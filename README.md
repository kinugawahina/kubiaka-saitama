# kubiaka-saitama

埼玉県のクビアカツヤカミキリ発生確認データを収集・整理し、現地調査向けに地図化するプロジェクト。

## 目的

* 埼玉県公式ArcGISデータから発生確認地点を取得する
* 年度別・市町村別・樹種別に発生状況を整理する
* 2024年度のホットスポットを抽出する
* 現地調査向けの地図を生成する
* 将来的に2025年以降の投稿データやX/Grok由来情報を重ね、2026年の発生予測に発展させる

## データ分類

ArcGISレイヤーでは年度によりフィールド名が異なるため、分析用に以下の正規化フィールドを追加している。

| 正規化フィールド           | 内容     | 元フィールド例           |
| ------------------ | ------ | ----------------- |
| `city_norm`        | 市町村名   | `市町村名`, `市町村`     |
| `tree_norm`        | 被害樹種   | `樹種`, `被害樹種`      |
| `adult_count_norm` | 成虫確認頭数 | `確認_成虫`, `成虫頭数`   |
| `larva_count_norm` | 幼虫確認頭数 | `確認_幼虫`, `幼虫頭数`   |
| `frass_norm`       | フラス確認  | `確認_フラス`, `フラス確認` |
| `date_found_norm`  | 確認日    | `発見年月日`, `県確知日`   |

## 現地調査方針

2024年度データでは、加須市・熊谷市・行田市・鴻巣市が高密度。

特に熊谷市では成虫確認頭数が非常に多い地点が複数あり、以下を優先候補とする。

* さくら運動公園
* 妻沼緑道
* 新奈良川遊水池
* 熊谷スポーツ文化公園
* 熊谷荒川緑地
* 石原かめの道
* 別府沼公園
* 荒川桜堤
