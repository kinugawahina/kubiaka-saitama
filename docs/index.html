埼玉県クビアカツヤカミキリ発生確認マップ分析

埼玉県の公開ArcGISデータをもとに、クビアカツヤカミキリの発生確認地点を整理・可視化したプロジェクトです。

注意: 本ページは公開データをもとにした個人の分析です。防除・通報・捕殺等は自治体の案内に従ってください。 個人宅等の詳細な探索を目的とした利用は避けてください。

年度別レイヤー切替マップ

2018〜2024年度の発生確認地点を年度別に表示・切替できる地図です。

2018→2024 拡散前線候補マップ

2024年度地点ごとに最も近い2018年度地点までの距離を計算し、 距離が大きい地点を拡散前線候補として表示した地図です。 公開版では個人宅地点を除外しています。
